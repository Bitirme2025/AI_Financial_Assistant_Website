Project Overview

This project presents the design and development of an AI-powered personalized financial assistant. The system enables retail investors to make informed decisions by integrating multiple AI modules: news-based sentiment analysis, stock & sector forecasting (TCN & MAMBA), and a Reinforcement Learning-based portfolio optimizer. Users interact with the system through an intuitive chat interface powered by a central AI agent.

Key Features

News Sentiment Analysis

Utilizes NLP transformer models (BERT/RoBERTa) to analyze multilingual financial news, determining sentiment direction (positive/negative) and impact on specific sectors.

Market Forecasting

Leverages Temporal Convolutional Networks (TCN) for individual stock price direction and MAMBA multi-scale attention models for long-term sector trend predictions.

RL Portfolio Optimizer

A Reinforcement Learning module that dynamically adapts asset allocation recommendations based on the user's risk profile, investment horizon, and market conditions.

RAG Document Assistant

Retrieval-Augmented Generation (RAG) allows users to upload financial reports and query them using natural language for context-aware insights.
